# New E-Course GACF

Laravel + Inertia + React + TypeScript landing page for Google Ads Cuan Formula.

## Routes

- `/`
- `/gacf-course-new-page-2026-bgs`

## Frontend

- Main page: `resources/js/pages/gacf-course.tsx`
- GACF components: `resources/js/components/landing/gacf`
- Landing copy/data: `resources/js/data/gacf-landing.ts`

## Environment

Set these values when the final checkout and tracking details are available:

```env
VITE_GACF_CHECKOUT_URL=
VITE_GACF_WHATSAPP_URL=
VITE_COURSE_PRICE=792000
VITE_META_PIXEL_ID=
VITE_CLARITY_ID=
```

Real testimonial and mentor assets are not included yet. The page intentionally renders labeled placeholders until official GACF assets are added.
